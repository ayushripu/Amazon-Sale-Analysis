# 📊 Amazon Sales Analysis

## 🚀 Project Overview

This project analyzes Amazon sales data using **Python (Pandas, Matplotlib)** and **MySQL**.
It focuses on data cleaning, transformation, visualization, and extracting meaningful business insights.

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* Matplotlib
* MySQL
* Jupyter Notebook

---

## 📂 Project Structure

```
Amazon-Sale-Analysis/
│
├── data/              # Dataset (CSV file)
├── notebooks/         # Jupyter Notebook (analysis)
├── sql/               # SQL queries
├── images/            # Visualizations (charts)
├── README.md
```

---

## 📊 Key Analysis Performed

### ✔️ Data Cleaning

* Removed null values
* Fixed column names
* Converted data types

### ✔️ Data Analysis

* Category-wise sales
* State-wise sales
* Order amount distribution

### ✔️ Data Visualization

* Pie chart (Top categories)
* Bar chart (Top states)
* Histogram (Order amount distribution)

---

## 📈 Key Insights

* 📌 **Top Category:** "Set" contributes the highest sales
* 📌 **Top States:** Maharashtra, Karnataka generate maximum revenue
* 📌 Most orders fall in a **specific price range (mid-range orders)**
* 📌 Few categories dominate overall sales
